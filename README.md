# emacs-compile-buffer
Compile buffer with single short-cut

# Installation

Copy `compile-buffer.el` to `~/.emacs.d/site-lisp`, and add following lines in `~/.emacs`:

    (add-to-list 'load-path "~/.emacs.d/site-lisp/")
    (load "compile-buffer.el")

# Usage

  * <F5> to compile
  * <F6> to run
